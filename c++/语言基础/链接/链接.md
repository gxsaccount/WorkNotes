# 编译 #
**目标：**   
生成**二进制可重定位**的目标文件 （.o）  
**阶段：**  
预编译：处理#号开头的命令（除#pragma lib/ #pragma link）    
编译：语法分析，词法分析，编译优化（-O...）     
汇编：转化机器吗,生成.o      

# 链接 #  
静态链接：  
1.将所有的**\.o文件段，**符号表合并后，进行符号解析**     
2.**符号的重定位**，得到可执行文件（a.out）
