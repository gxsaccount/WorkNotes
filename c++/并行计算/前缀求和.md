前缀求和问题:  
给出x0,x1,x2,...,xn-1,求出所有部分和(x0+x1,x0+x1+x2,...)  
  
  sum[0] = x[0];
  for(int i = 1; i < n; ++i){
    sum[i] = sum[i-1] + x[i];
  }
  
  for(j = 0;j < log(n); ++j){
      forall(i = 2^j; i < n; ++i ){
          if (i >= 2^j) x[i] = x[i] + x[i - 2^j];
      }
  }
